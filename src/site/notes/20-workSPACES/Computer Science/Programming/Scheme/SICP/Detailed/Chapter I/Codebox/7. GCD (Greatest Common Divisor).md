# 7. GCD (Greatest Common Divisor)

## Code
```scheme
#lang racket


(define (gcd a b)
  (if (= b 0)
      a
      (gcd b (remainder a b))))


(require rackunit)
(require rackunit/text-ui)

(define gcd-tests
  (test-suite
   "GCD TESTS"
   (test-case
    "Perfect Divisors"
    (check-equal? (gcd 14 7) 7)
    (check-equal? (gcd 27 9) 9))

   (test-case
    "Random"
    (check-equal? (gcd 24 14) 2)
    (check-equal? (gcd 36 24) 12))

   (test-case
    "Primes"
    (check-equal? (gcd 15 14) 1)
    (check-equal? (gcd 34 11) 1)))
  )

(run-tests gcd-tests)
```
